Detailed Description

An ephemeral type-erased way to store all the metadata necessary to reconstruct any given parameter object. You can use this type to save and later recreate a parameter object.

See also
ParameterLayout, ParameterList, Parameter, TypedParameter

◆ createParameter()

std::unique_ptr<Parameter> lemons::plugin::ParameterTraits::createParameter ( const ParameterList &  list) const

Creates a parameter object of the correct type for this ParameterTraits object.

If this ParameterTraits represents a meta parameter, then the returned object will be a kind of MetaParameter. If this ParameterTraits represents a meter parameter, then the returned object will be a kind of MeterParameter. Otherwise, the returned object will be a kind of TypedParameter. The parameter's value type is determined by the ValueType enum.
