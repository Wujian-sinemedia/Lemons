Detailed Description

template<typename StateType>
class lemons::plugin::GUI< StateType >

Base class for plugin GUI's.

Your plugin's main component should inherit from this.

Template Parameters
StateTypeThe type of your plugin's main state object. This should be the same StateType that you sent to Processor and PluginState's template arguments.
