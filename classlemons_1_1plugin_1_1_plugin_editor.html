Detailed Description

template<typename ContentComponentType, typename StateType>
class lemons::plugin::PluginEditor< ContentComponentType, StateType >

Base class for a plugin editor that instantiates the owned content for you.

Template Parameters
ContentComponentTypeThe type of your plugin's main component. This must inherit from GUIBase, but should preferably inherit from GUI<StateType>.
This type's constructor signature must be ContentComponentType (PluginState< StateType >&).
StateTypeThe type of your plugin's main state object.
