Detailed Description

template<typename ContentComponentType, LEMONS_MUST_INHERIT_FROM(ContentComponentType, juce::Component)>
class lemons::plugin::Editor< ContentComponentType, >

Base class for a plugin editor that simply holds a specified GUI component.

Template Parameters
ContentComponentTypeThe type of your plugin's main component. This type must inherit from GUI<StateType>.
StateTypeThe type of the plugin's state object. This type must inherit from State.
