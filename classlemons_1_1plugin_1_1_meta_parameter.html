parameterCategoryAn optional parameter category. See juce::AudioProcessorParameter::Category.

float  minAmt,
float  maxAmt 
)
inherited

Creates or edits a connection to a given parameter.

If a connection to the given parameter already exists, it will be remapped to the new min & max amounts; otherwise, a new connection will be created. The min and max values should be between 0.f and 1.f, and can be thought of as the percent effect this meta parameter will have on this parameter.
