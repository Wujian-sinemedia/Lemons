◆ FileType

enum lemons::files::FileType
strong

An enum used to describe various kinds of binary encodings of serialized data.

Enumerator
XML 

Represents an XML file.

JSON 

Represents a JSON file.

Opaque 

Represents any non-standard/proprietary method of encoding and decoding information to/from binary.

May not be human-readable.
