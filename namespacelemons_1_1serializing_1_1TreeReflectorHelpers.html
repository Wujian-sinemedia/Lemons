◆ makePropertyNameForElement()

String lemons::serializing::TreeReflectorHelpers::makePropertyNameForElement ( const String &  propertyName,
int &  index 
)

Transforms a passed generic container element name and container index into a unique name for a given object.

The transformation is: 

{propertyName}_{index}

For example, passing "Item" with an index of 3 will return "Item_3".

This function also increments the referenced index each time it's called.

Referenced by addContainer().

◆ propertyNameToContainerName()

String lemons::serializing::TreeReflectorHelpers::propertyNameToContainerName ( const String &  propertyName)

Transforms the name of a single container element to a name for the container's sub-node.

This simply appends an "s" to the passed propertyName.
For example, passing "Item" will return "Items".
