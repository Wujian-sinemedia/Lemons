Detailed Description

This class represents the data of an audio file, including the AudioBuffer itself, as well as the samplerate, bit depth, and metadata information.

One of these can be created from an audio file on disk, or from any kind of juce::InputStream.

◆ getData()

template<typename SampleType >
const AudioBuffer<SampleType>& lemons::AudioFile::getData ( )

Returns the actual audio data of this audio file.

If isValid() returns false, then this function will return a buffer with a size of 0 channels and 0 samples. Internally, the data is always loaded as floats; the first time you call the double version of this function, the data will be converted to double precision internally.
