Detailed Description

template<typename SampleType>
class lemons::dsp::psola::Analyzer< SampleType >

Performs analysis of the incoming signal, to prepare for the Shifter's resynthesis.

This detects the pitch of the input signal and splits the incoming audio frame into analysis grains, ready to be respaced by Shifter objects.
This is meant to be used with perfectly monophonic signals.

See also
Shifter, dsp::PitchDetector
