Converts a MIDI file object to a MIDI buffer.

Parameters
fileThe MIDI file object to read from.
trackToReadIf this is greater than -1, then only events from the specified track number will be added to the returned buffer. If this is -1, then all events from every track of the MIDI file will be concatenated into the output buffer.
