◆ getStrings()

juce::StringArray lemons::binary::getStrings ( const String &  textFileName)

Returns an array of strings, each containing a line of a text file in the BinaryData target.

This loads the file as a string and parses it into tokens using line break and carriage return characters. If the file can't be loaded, an assertion will be thrown.

constexprnoexcept

Returns true if this module was compiled with the LEMONS_HAS_BINARY_DATA flag set to 1.
