# Instructions for building Doxygen documentation for Lemons

## Dependencies

- doxygen
- python
- make
- graphviz (to generate inheritance diagrams)

Make sure that all the dependencies can be found in your PATH.

## Building

- cd into the `doxygen` directory on the command line
- run `make`

Doxygen will create a new subdirectory "doc". 
Open doc/index.html in your browser to access the generated HTML documentation.
