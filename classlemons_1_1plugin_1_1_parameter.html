Detailed Description

A plugin parameter class that allows you to use my Serializing API, and has a few other cool bells and whistles.

To actually use one, I recommend you stick it inside a ParameterHolder, to make your life easier accessing it once you've added it to your AudioProcessor.

See also
ParameterHolder, ParameterList, TypedParameter

paramNameThe name of this parameter.
paramRangeThe range of values that this parameter can have.
paramDefaultValueThe default value of this parameter.
valueToTextFuncToUseAn optional lambda function that converts this parameter's value to a text description.
textToValueFuncToUseAn optional lambda function that converts a text input string to a value for this parameter.
paramLabelAn optional label to use for this parameter's units.
isAutomatableBoolean flag that indicates whether this parameter should be automatable in the user's DAW.
metaParamBoolean flag that indicates whether this parameter represents a "meta-parameter" – that is, a parameter that controls other parameters.
parameterCategoryAn optional parameter category. See juce::AudioProcessorParameter::Category.
