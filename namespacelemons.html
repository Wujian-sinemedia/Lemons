◆ stringToPitch()

int lemons::stringToPitch ( const String &  string)
noexcept

Converts a string description of a pitch to a MIDI note number.

The string should be in the format 'A#7', 'Bb9', etc. You can also use the Unicode sharp, flat, and natural symbols in the passed string. If the passed string is in the wrong format, or it can't be parsed correctly for some reason, this will return -1.

See also
stringToPitchClass(), pitchToString()
