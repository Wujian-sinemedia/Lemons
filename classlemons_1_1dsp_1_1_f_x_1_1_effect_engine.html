Detailed Description

template<template< typename FloatType > class EffectType, typename SampleType, LEMONS_MUST_INHERIT_FROM(EffectType< SampleType >, AudioEffect< SampleType >)>
class lemons::dsp::FX::EffectEngine< EffectType, SampleType, >

A class that wraps an AudioEffect into a dsp::Engine.

Template Parameters
EffectTypeA template class representing the type of effect this engine will manage – for example, EQ<SampleType> or Compressor<SampleType>.
SampleTypeThe specialization of EffectType that this engine will instantiate. Must be float or double.
See also
AudioEffect, dsp::Engine
