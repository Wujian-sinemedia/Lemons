Detailed Description

template<typename SampleType>
class lemons::dsp::Engine< SampleType >

Base class for an audio engine. This should be a self-contained signal chain. This class handles pop-preventing between succesive bypassed/unbypassed frames, and provides several options for top-level process() function APIs while allowing you to implement only one internal renderBlock() function.
