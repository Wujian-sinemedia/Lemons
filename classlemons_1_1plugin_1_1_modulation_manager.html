Detailed Description

A modulation manager for a plugin. This class holds a collection of LFOs that can be mapped to any number of Parameter objects.

◆ getLFO()

LFO* lemons::plugin::ModulationManager::getLFO ( int  index)

Returns the LFO at the given index in the modulation manager's vector of LFOs.

The returned pointer may be null if an LFO does not exist at the requested index.

This is the least preferable way to retrieve LFO objects; prefer to use getLFOofType(), getLFOatFrequency(), or getLFOwithConnection().
