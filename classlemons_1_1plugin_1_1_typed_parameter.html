Detailed Description

template<typename ValueType>
class lemons::plugin::TypedParameter< ValueType >

A parameter that can be stored as a type other than a float, but still inherits from my Parameter base class.

Template Parameters

Creates a typed parameter.

This constructor allows you to specify the minimum and maximum values, instead of supplying a range object.

Parameters
minimumThe minimum value the parameter can have.
maximumThe maximum value the parameter can have.
defaultValueThe initial default value for the parameter.
paramNameThe name of this parameter.
stringFromValueAn optional lambda function that converts this parameter's value to a text description.
valueFromStringAn optional lambda function that converts a text input string to a value for this parameter.
paramLabelAn optional label to use for this parameter's units.
isAutomatableBoolean flag that indicates whether this parameter should be automatable in the user's DAW.
metaParamBoolean flag that indicates whether this parameter represents a "meta-parameter" – that is, a parameter that controls other parameters.
parameterCategoryAn optional parameter category. See juce::AudioProcessorParameter::Category.
