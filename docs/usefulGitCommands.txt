
### Delete a git submodule ###

First, delete the relevant entry from the .gitmodules file

git rm --cached /path/to/submodule

rm -rf .git/modules/path/to/submodule

git commit -m "Removed submodule" -a
git push



### Update git submodules recursively ###

git pull

git submodule foreach 'git fetch && git checkout origin/HEAD && git submodule update --recursive --merge'

git commit -m "submodule update" -a
git push
