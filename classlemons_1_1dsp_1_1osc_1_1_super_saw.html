Detailed Description

template<typename SampleType>
class lemons::dsp::osc::SuperSaw< SampleType >

A basic "super saw" oscillator.

This oscillator contains 7 slightly detuned saw oscillators and returns their aggregate output.
You can set the detuning amount in cents with setDetuneAmount().
