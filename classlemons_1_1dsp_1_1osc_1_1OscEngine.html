Detailed Description

template<typename SampleType, template< typename T > class Osctype>
class lemons::dsp::osc::OscEngine< SampleType, Osctype >

A utility class that wraps an Oscillator into a self-contained dsp::Engine.

This will instantiate and manage the appropriate oscillator type for you.
Example usage:

using namspace lemons::dsp::osc;
auto sineEngine = OscEngine<float, Sine>;
Template Parameters
SampleTypeThe floating point type to be used for samples. Must be either float or double.
OsctypeA template class representing the kind of oscillator you want this engine to use; for example, osc::Sine. This should be un-specialized. This type must inherit from Oscillator.
See also
Oscillator, dsp::Engine
