Detailed Description

template<typename SampleType>
class lemons::dsp::AudioFifo< SampleType >

A multichannel audio FIFO.

If you only need to store a single channel's worth of audio samples, consider using the CircularBuffer class instead. Also note that while the CircularBuffer class is thread-safe, this class is not.

See also
CircularBuffer, AudioAndMidiFIFO
