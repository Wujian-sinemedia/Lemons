#define LEMONS_MUST_BE_SPECIALIZATION (   classToTest,
  requiredTemplate 
)    std::enable_if_t<lemons::is_specialization<classToTest, requiredTemplate>::value>* = nullptr

Use this macro inside a template declaration to ensure that classToTest is a specialization of the required template. For example:

template < class VectorType, LEMONS_MUST_BE_SPECIALIZATION(VectorType, std::vector) >
class SomeClass { };

With the above code, instantiating

SomeClass< std::vector<int> >

will succeed, and attempting to instantiate

SomeClass< std::list<int> >;

will throw a compile-time error.
Note that the first argument to this macro should be a fully-specialized type, and the second argument to this macro must be an unspecialized template!

◆ LEMONS_MUST_INHERIT_FROM

#define LEMONS_MUST_INHERIT_FROM (   classToTest,
  requiredBaseClass 
)    std::enable_if_t<std::is_base_of<requiredBaseClass, classToTest>::value>* = nullptr

Use this macro inside a template declaration to place inheritance constraints on a template argument. For example:

template < class ParameterType, LEMONS_MUST_INHERIT_FROM(ParameterType, juce::AudioProcessorParameter) >
class SomeClass { };

With the above code, attempting to instantiate a specialization of SomeClass where the template argument is not a type that inherits from juce::AudioProcessorParameter will throw a compile-time error.
Note that both arguments to this macro must be fully specialized types.
